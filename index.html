GeoHunt App for Android

Date: Sept - Dec 2025

Created for: COMS 3090 - Software Development Practices

This project involved creating an Android application that utilizes a web server and database using Java and Springboot, and the HTTP and Websocket protocols. My role in the development of the app involved backend/database programming and design, as part of a larger team of 4.

Users are tasked with finding nearby real world locations based on Google Street View images. By playing the game, users can earn points to buy power-ups and customization options for their profile. There are also social features, allowing users to compete and interact with friends.
